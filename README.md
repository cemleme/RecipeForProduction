# recipes for production

Recipes for Production is a tool for food producers to prepare recipes and easily get the nutrition facts.
The user can add unlimited ingredients with their nutrition facts (calorie, protein, salt, fat etc.) and use these ingredients to form a recipe for a product.
A product might contain multiple recipes to achieve perfect keeping old recipes as it's history as well.
The final recipe shows it's own nutrition facts gathering the data from all the ingredients used in their proportions.
